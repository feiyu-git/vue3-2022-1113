### 常见问题：

##### 1、[【github】git push错误failed to push some refs to的解决](<https://blog.csdn.net/rocling/article/details/82956402>)

解决方案：

​		这个问题是因为远程库与本地库不一致造成的，那么我们把远程库同步到本地库就可以了。
使用指令。

```
git pull --rebase origin master
```
